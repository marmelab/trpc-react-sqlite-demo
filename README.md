# tRPC + React + Express

Demo of using tRPC with React front-end and Express back-end.

## Setup the back-end

```sh
cd server
cp .env.example .env
npm install
```

## Setup the front-end

```sh
cd client
npm install
```

## Run the back-end

```sh
cd server
npm run dev
```

## Run the front-end

```sh
cd client
npm run dev
```
